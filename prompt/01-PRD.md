# Product Requirements Document

# Portfolio v2

Version: 1.0

---

# Overview

Portfolio v2 is a premium personal website showcasing professional experience, AI engineering capabilities, product thinking, and selected projects through a modern, interactive interface.

The portfolio should resemble a polished SaaS landing page rather than a traditional resume website.

---

# Objectives

- Showcase professional work.
- Demonstrate frontend engineering skills.
- Present AI expertise.
- Increase recruiter engagement.
- Create a memorable user experience.
